# Feedback Collector

Simple feedback collector demo using React + Vite.

Run locally:

1. Install dependencies

```powershell
cd "c:\Users\DELL\OneDrive\Desktop\feedBackCollector"; npm install
```

2. Start dev server

```powershell
npm run dev
```

This app stores feedback in `localStorage` so entries persist in your browser.

Files of interest:
- `src/components/FeedbackForm.jsx`
- `src/components/FeedbackList.jsx`
- `src/components/FeedbackItem.jsx`
- `src/services/FeedbackService.js`
